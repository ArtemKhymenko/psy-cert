# psy-cert.com.ua

Заглушка для майбутньої інформаційної платформи про сертифікацію психологів в Україні (Закон № 4223-IX). Запуск — осінь 2026.

## Стек
- Чистий HTML/CSS/JS, без збірок
- Hosting: Cloudflare Pages
- Email-форма: Web3Forms
- Шрифт: Manrope (Google Fonts)

## Структура
```
/
├── index.html       — головна сторінка
├── robots.txt       — інструкції для пошукових ботів
├── sitemap.xml      — карта сайту
└── README.md
```

## Перед деплоєм
У `index.html` замінити `REPLACE_WITH_YOUR_WEB3FORMS_ACCESS_KEY` на реальний ключ із web3forms.com.
